# StringRules

StringRules describe the constraints applied to `string` values

This message contains a oneof named well_known. Only a single field of the following list may be set at a time:
  - email
  - hostname
  - ip
  - ipv4
  - ipv6
  - uri
  - uriRef
  - address
  - uuid
  - wellKnownRegex
